# Contributing to APSMMB

## Who Can Contribute

- APSMMB members in good standing
- Appointed officers and committee members
- Invited collaborators approved by the Executive Committee

## How to Contribute

1. Fork the repository
2. Create a new branch: `git checkout -b feature/your-feature-name`
3. Make your changes
4. Commit: `git commit -m "Add: description of change"`
5. Push and submit a Pull Request

## Style Guidelines

- Use clear, professional, and academically appropriate language
- Follow existing file naming conventions (lowercase, hyphenated)
- Do not upload personal data or unpublished manuscripts without permission
- All images must be original, licensed, or CC BY compatible
